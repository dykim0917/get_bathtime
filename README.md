# get_bathtime

Bath Time landing and policy pages.

## Pages

- `/` landing page
- `/privacy` privacy policy
- `/terms` terms of service
- `/support` support contact

## Domain Plan

- `getbathtime.com` -> this landing site
- `www.getbathtime.com` -> this landing site
- `app.getbathtime.com` -> the Expo web app

Landing CTAs point to `https://app.getbathtime.com`.

## Commands

```bash
npm install
npm run dev
npm run build
```
